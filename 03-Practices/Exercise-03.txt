Exercise 3 - Arrays & Structs
-----------------------------

Create a program that allows to persist the course attendant information.
Use structures in order to group attendants related information (name, surname, age).
Create another struc called "course" that contains an array of attendants (maximum number of attendants set by a define) and the starting date.
Create a function that allows to load data.
Create another function that allows to print all attendants.
