Exercise 4a - Pointers
----------------------

Create an array in order to storage a 20 chars string.
Create a function that allows to fill the variable using the keyboard.
Create another function in order to receive an string as reference and then print it using putch function.


Exercise 4b - Pointers
----------------------

Create a function that scans the string and replace all numbers (1 digit) by the same number less 1 (floor 0).

Example 1: Input = Hi, mi age is 35 -> Output = Hi mi age is 24
Example 2: Input = I have 230 euros -> Output = I have 120 euros
